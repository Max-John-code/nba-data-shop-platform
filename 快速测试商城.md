## 快速测试步骤

### 1. 确认服务运行
后端服务已在运行：http://127.0.0.1:8000
前端需要启动（如果还没启动）：
```bash
cd nba-data-shop-platform
npm run dev:h5
```

### 2. 测试管理员添加商品
1. 打开浏览器访问前端地址
2. 登录管理员账号（验证码：9999）
3. 点击右上角"管理员" → 进入管理后台
4. 点击"🛍️ 商品管理"
5. 点击"添加商品"
6. 填写信息并上传图片
7. 提交保存

### 3. 测试用户购买流程
1. 首页点击"球星商城"
2. 浏览商品列表
3. 点击商品查看详情
4. 点击"加入购物车"
5. 点击右上角🛒查看购物车
6. 点击"去结算"
7. 填写收货信息
8. 提交订单
9. 在个人中心查看"我的订单"

## 系统已有3个示例商品
- 詹姆斯23号球衣 ¥299
- 库里30号球衣 ¥289  
- 科比24号纪念球衣 ¥399

## 所有入口位置
- 首页导航 → "球星商城"
- 个人中心 → "我的订单"（📦图标）
- 管理后台 → "商品管理"（🛍️图标）
- 商城页面 → 右上角🛒购物车
