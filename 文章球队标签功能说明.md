# 文章球队标签功能说明

## 功能概述
为文章添加球队标签，用于后续的协同过滤推荐算法。系统会根据用户点赞的球队文章，优先推荐该球队的相关内容。

## 数据库更新

### 1. 执行SQL脚本
```bash
cd nba_backend
添加文章球队字段.bat
```

### 2. 字段说明
- **表名**: `article`
- **新增字段**: `team` (VARCHAR(50))
- **说明**: 存储文章所属的球队名称

## 后端更新

### 1. 模型更新 (`forum/models.py`)
```python
class Article(models.Model):
    team = models.CharField(max_length=50, blank=True, default='', verbose_name='所属球队')
```

### 2. 序列化器更新 (`forum/serializers.py`)
- `ArticleSerializer`: 添加 `team` 字段
- `ArticleDetailSerializer`: 添加 `team` 字段

## 前端更新

### 1. 管理端 - 文章表单 (`admin/article-form.vue`)
- 添加球队选择器（Picker组件）
- 支持30个NBA球队选择
- 发表文章时必须选择球队

### 2. 管理端 - 文章列表 (`admin/articles.vue`)
- 显示球队标签（紫色渐变徽章）
- 标签位置：标题右侧

### 3. 用户端 - 文章列表 (`forum/index.vue`)
- 显示球队标签
- 标签样式：紫色渐变，圆角徽章

### 4. 用户端 - 文章详情 (`forum/detail.vue`)
- 标题旁显示球队标签
- 与点赞收藏按钮配合显示

## 球队列表
支持30个NBA球队：
- 西部：湖人、勇士、快船、国王、太阳、独行侠、火箭、马刺、雷霆、灰熊、森林狼、掘金、开拓者、爵士、鹈鹕
- 东部：凯尔特人、篮网、尼克斯、76人、猛龙、公牛、骑士、活塞、步行者、雄鹿、老鹰、黄蜂、热火、魔术、奇才

## 推荐算法准备
此功能为协同过滤推荐算法做准备：
1. 记录用户点赞的文章及其球队标签
2. 统计用户对各球队的兴趣度
3. 根据兴趣度调整文章推荐顺序
4. 优先展示用户感兴趣球队的文章

## 精彩回放
精彩回放已有 `teams` 字段（对阵球队），格式如："湖人 vs 勇士"

## 使用流程

### 管理员发表文章
1. 进入"交流论坛管理"
2. 点击"发表文章"
3. 填写标题
4. 选择所属球队（必填）
5. 填写内容
6. 上传图片（可选）
7. 点击"发表"

### 用户浏览文章
1. 进入"交流论坛"
2. 查看文章列表，每篇文章显示球队标签
3. 点击文章查看详情
4. 点赞/收藏感兴趣的文章
5. 系统记录用户对该球队的兴趣

## 注意事项
1. 球队字段为必填项
2. 现有文章的球队字段默认为空，需要管理员编辑补充
3. 球队标签使用紫色渐变样式，与系统主题一致
4. 推荐算法将在后续版本实现
