## 测试流程

### 1. 管理员添加商品
1. 使用管理员账号登录（手机号+验证码9999）
2. 点击右上角"管理员"进入管理后台
3. 点击"🛍️ 商品管理"
4. 点击"添加商品"按钮
5. 填写商品信息：
   - 商品名称：例如"詹姆斯23号球衣"
   - 商品描述：详细描述
   - 上传商品图片
   - 价格：例如299.00
   - 库存：例如100
   - 分类：选择"球衣"
   - 关联球星：例如"勒布朗詹姆斯"
6. 点击"提交"保存

### 2. 用户浏览商品
1. 在首页点击"球星商城"导航
2. 查看商品列表（显示所有上架商品）
3. 可以按分类筛选：全部/球衣/球鞋/帽子/其他
4. 点击商品卡片查看详情

### 3. 加入购物车
1. 在商品详情页
2. 选择购买数量（默认1）
3. 点击"加入购物车"按钮
4. 提示"已加入购物车"

### 4. 查看购物车
1. 在商城首页点击右上角"🛒"图标
2. 查看购物车中的商品
3. 可以修改数量（+/-按钮）
4. 可以删除商品（点击"删除"）
5. 查看总价

### 5. 结算下单
1. 在购物车页面点击"去结算"
2. 填写收货信息：
   - 收货人姓名
   - 收货电话
   - 收货地址
3. 确认订单信息和总金额
4. 点击"提交订单"
5. 提示"订单创建成功"

### 6. 查看订单
1. 方式一：在个人中心点击"我的订单"
2. 方式二：在商城页面点击右上角"📋"图标
3. 查看订单列表：
   - 订单号
   - 订单状态（待支付/已支付/已发货/已完成/已取消）
   - 订单金额
   - 创建时间
4. 点击订单查看详情（商品列表、收货信息）

## 注意事项

1. 必须登录后才能使用购物车和下单功能
2. 商品库存不足时无法加入购物车
3. 订单创建后会自动扣减库存、增加销量
4. 管理员可以在后台管理所有商品（添加/编辑/删除/上下架）
5. 商品图片使用base64存储，支持大图片（LONGTEXT类型）

## 入口位置

### 管理端入口
- 管理后台主页 → "🛍️ 商品管理"

## API接口

### 商品相关
- GET /api/shop/products/ - 获取商品列表
- GET /api/shop/products/:id/ - 获取商品详情
- POST /api/shop/products/ - 添加商品（管理员）
- PUT /api/shop/products/:id/ - 更新商品（管理员）
- DELETE /api/shop/products/:id/ - 删除商品（管理员）

### 购物车相关
- GET /api/shop/cart/ - 获取购物车
- POST /api/shop/cart/ - 添加到购物车
- PUT /api/shop/cart/:id/ - 更新购物车数量
- DELETE /api/shop/cart/:id/ - 删除购物车项

### 订单相关
- GET /api/shop/orders/ - 获取订单列表
- POST /api/shop/orders/ - 创建订单
- GET /api/shop/orders/:id/ - 获取订单详情
