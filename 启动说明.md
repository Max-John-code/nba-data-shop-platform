# NBA数据商城平台 - 启动说明

## 后端启动步骤

1. 进入后端目录
```bash
cd nba_backend
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 数据库迁移
```bash
python manage.py makemigrations
python manage.py migrate
```

4. 启动后端服务
```bash
python manage.py runserver
```

后端将在 http://127.0.0.1:8000 运行

## 前端启动步骤

1. 进入前端目录
```bash
cd nba-data-shop-platform
```

2. 安装依赖（如果还没安装）
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev:h5
```

## 功能说明

### 已实现功能
- ✅ 用户注册（手机号+验证码）
- ✅ 用户登录（手机号+验证码）
- ✅ 短信验证码发送（开发环境会在toast中显示验证码）
- ✅ Token认证
- ✅ 用户协议勾选验证

### API接口
- POST `/api/accounts/send-sms/` - 发送验证码
- POST `/api/accounts/login/` - 登录
- POST `/api/accounts/register/` - 注册

### 开发环境说明
- 验证码在开发环境会直接返回并显示在toast中，方便测试
- 验证码有效期：5分钟
- 验证码发送间隔：60秒
- 验证码格式：4位数字

## 测试流程

1. 启动后端服务
2. 启动前端服务
3. 在前端页面输入手机号（任意11位数字）
4. 点击"获取验证码"，验证码会显示在toast提示中
5. 输入验证码
6. 勾选用户协议
7. 点击"注册"或"登录"按钮
