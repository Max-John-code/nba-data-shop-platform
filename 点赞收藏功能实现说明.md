### 2. API接口

#### 视频相关
```
POST   /api/highlights/{id}/like/          # 点赞视频
DELETE /api/highlights/{id}/like/          # 取消点赞
POST   /api/highlights/{id}/favorite/      # 收藏视频
DELETE /api/highlights/{id}/favorite/      # 取消收藏
GET    /api/highlights/{id}/status/        # 获取点赞/收藏状态
```

#### 文章相关
```
POST   /api/forum/articles/{id}/like/      # 点赞文章
DELETE /api/forum/articles/{id}/like/      # 取消点赞
POST   /api/forum/articles/{id}/favorite/  # 收藏文章
DELETE /api/forum/articles/{id}/favorite/  # 取消收藏
GET    /api/forum/articles/{id}/status/    # 获取点赞/收藏状态
```

#### 用户收藏列表
```
GET /api/accounts/favorites/highlights/    # 我收藏的视频
GET /api/accounts/favorites/articles/      # 我收藏的文章
GET /api/accounts/likes/highlights/        # 我点赞的视频
GET /api/accounts/likes/articles/          # 我点赞的文章
```

## 🎨 前端UI设计建议

### 按钮样式
```vue
<button class="like-btn" :class="{ active: isLiked }" @click="toggleLike">
  <text class="icon">{{ isLiked ? '❤️' : '🤍' }}</text>
  <text class="count">{{ likes }}</text>
</button>

<button class="favorite-btn" :class="{ active: isFavorited }" @click="toggleFavorite">
  <text class="icon">{{ isFavorited ? '⭐' : '☆' }}</text>
  <text class="count">{{ favorites }}</text>
</button>
```

### 交互效果
- 点击时有缩放动画
- 点赞成功后数字+1并有弹跳效果
- 取消时数字-1
- 按钮颜色变化(未点赞灰色,已点赞红色/黄色)
